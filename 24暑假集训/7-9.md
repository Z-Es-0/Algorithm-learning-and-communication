## 这是一道完整的dp题目

状态表示，dp[i]表示到目前以s[i]为结尾的最大长度子序列；

状态转移：dp[i]=dp[index of (s[i]-1)]+1； 即s[i]-1的答案+1;

考虑用map记录s[i]-1的索引值；

可以写成简单的dp code


```cpp
 for (int i = 0; i < n; ++i) {
        mp[s[i]]=i;
        if (mp.count(s[i]-1)>0){
            dp[i]=dp[mp[s[i]-1]]+1;
            
        } else{
            dp[i]=1;
         
        }
    }
```
