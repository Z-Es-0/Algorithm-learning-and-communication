# 二叉树

## 二叉树的遍历

### 前序遍历

先遍历根节点，然后遍历左孩子，右孩子

### 中序遍历

先遍历左节点然后访问根节点最后右节点

### 后序遍历

先左孩子，然后遍历右孩子，最后访问根节点


### 给出中序后序遍历求前序遍历

```python

def build(s1,s2):
    if len(s2)<=0:
        return
    root=s2[-1]
    if len(s1)==1:
        print(s1[0],end='')
        return
    for i in range(len(s1)):
        if s1[i]==root:
            print(s1[i],end='')
            s1left=s1[:i]
            s1right=s1[i+1:]
            break
    s2left,s2right=[],[]
    for i in s2:
        if i in s1left:
            s2left.append(i)
        elif i in s1right:
            s2right.append(i)
    build(s1left,s2left)
    build(s1right,s2right)

if __name__ == '__main__':
    s1 = list(input())
    s2 = list(input())
    build(s1,s2)
```
